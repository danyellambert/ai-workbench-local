# Contributing

Thanks for checking out Axiovance.

Axiovance is currently maintained as a curated product project. Feedback, bug reports and thoughtful suggestions are welcome.

## Good ways to help

- Report bugs or broken flows
- Point out unclear documentation
- Suggest product or workflow improvements
- Share ideas for better deployment or developer experience

## Pull requests

Small, focused pull requests are welcome.

Before opening a pull request, please avoid committing:

- secrets or credentials
- local `.env` files
- generated runtime data
- build artifacts

For larger changes, opening an issue first is preferred.
