# Executive Deck Generation — slide recipes by deck type

## Objective

Define the standard slide sequence for each priority deck type.

---

## P1 — Benchmark & Eval Executive Review

Recommended sequence:

1. title
2. executive summary
3. key metrics
4. model leaderboard
5. eval suite leaderboard
6. recommendation vs watchouts
7. next steps

---

## P2 — Document Review Deck

Recommended sequence:

1. title
2. executive summary
3. risk snapshot metrics
4. top findings table
5. evidence / implications
6. recommendation vs watchouts
7. action plan / next steps

---

## P2 — Policy / Contract Comparison Deck

Recommended sequence:

1. title
2. comparison summary
3. comparison table
4. impact analysis
5. recommendation vs watchouts
6. next steps

---

## P3 — Action Plan Deck

Recommended sequence:

1. title
2. executive summary
3. priority metrics
4. workstreams / cards
5. action table
6. timeline
7. blockers vs mitigations

---

## P3 — Candidate Review Deck

Recommended sequence:

1. title / candidate snapshot
2. executive summary
3. strengths
4. gaps / risks
5. evidence highlights table
6. recommendation vs watchouts
7. next steps

---

## P3 — Evidence Pack Deck

Recommended sequence:

1. title
2. executive summary
3. evidence metrics
4. findings table
5. owners / status table
6. risks vs mitigations
7. next steps
