# benchmark_runs

Local benchmark outputs live here.

This directory is intentionally ignored by Git except for this README and/or `.gitkeep`.

Do not commit generated benchmark outputs, Chroma stores, SQLite files, `.bin` index files, logs, reports, or `.DS_Store` files from this directory.

Current product and evaluation tooling may read benchmark bundles from local, baseline, or runtime data roots, but generated run artifacts should be treated as local/runtime data rather than source code.
