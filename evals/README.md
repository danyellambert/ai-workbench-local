# Evals and benchmark workspace

This directory contains tracked evaluation fixtures, benchmark configuration, and lightweight benchmark directory documentation.

The root-level directories below are local-only workspaces and are ignored by Git:

- `phase5_eval/`
- `phase8_eval/`
- `benchmark_pdfs/`
- `benchmark_runs/`

Tracked source-like evaluation inputs live here instead:

- `evals/phase5/fixtures/`
- `evals/phase8/configs/`
- `evals/phase8/fixtures/`
- `evals/benchmark-pdfs/`
- `evals/benchmark-runs/`

Generated reports, benchmark outputs, local PDF corpora, caches, `.DS_Store`, and heavy local artifacts should not be committed.
