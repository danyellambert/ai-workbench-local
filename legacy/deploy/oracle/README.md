# Oracle deploy assets

This directory preserves Oracle-specific host-level deployment assets.

These files are not part of the current AWS deployment path.

Current AWS deployment uses the generic deployment bundle, `.env.aws`, `docker-compose.aws.yml`, `scripts/deploy_aws.sh`, and `scripts/smoke_aws.sh`.

`Caddyfile.example` is retained here as a historical/deferred Oracle Always Free reverse-proxy template.
